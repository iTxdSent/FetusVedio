# 03-Vue前端壳层.SPEC.md

## 目标
完成中文界面的 Vue 前端壳层，能够展示视频、状态、模拟分类结果与模拟测量值。

## 环境约束（新增）
1. 前端联调所依赖的后端服务，必须运行在 Miniconda 的 Python 3.10 环境中。

## 本阶段数据来源
全部依赖后端 WebSocket 和 REST 提供的数据，不关心真实模型。

## 必做页面
1. 患者信息页
2. 实时测量页
3. 历史记录页

## 实时测量页要求
1. 左侧视频区域
2. 右侧信息区域
3. 中文按钮：
   - 开始检查
   - 结束检查
   - 冻结
   - 解冻
   - 选择本地视频
   - 恢复视频流
   - 手动保存
4. 实时显示：
   - 当前切面
   - 分类置信度
   - 测量值
   - 自动留图状态

## UI 说明
1. 面向中文用户，界面文案全部中文
2. 可用组件库，但风格简洁，不要过度设计
3. 不要因为等待真实模型而阻塞页面开发

## 验收标准
1. 前端在接入 Mock 数据时可完整展示流程
2. 中文文案完整
3. 视频与测量信息联动正常
