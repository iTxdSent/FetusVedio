# 05-数据持久化、自动留图与历史查询.SPEC.md

## 目标
在 Mock 数据驱动下，跑通数据库、文件保存、自动留图和历史查询链路。

## 环境约束（新增）
1. 本阶段后端执行环境固定为 Miniconda Python 3.10。
2. 与数据持久化相关的新增依赖需要同步维护到 conda 环境定义文件。

## 本阶段模型说明
本阶段仍使用 Mock 结果作为自动留图触发依据。

## 自动留图规则
1. plane != OTHER
2. measurement.success = true
3. 必填测量字段完整
4. 连续 2 帧满足条件
5. 同一切面 5 秒内不重复保存

## 必做内容
1. patients / sessions / measurements / snapshots 表
2. 自动保存 raw / overlay / result.json
3. 手动保存
4. 历史记录查询页面
5. 中文状态提示

## 验收标准
1. 即使没有真实模型，也能形成完整历史记录
2. 自动留图逻辑完全依赖统一结果契约，而不是依赖真实模型实现细节
